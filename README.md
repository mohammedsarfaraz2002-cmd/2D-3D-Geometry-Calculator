# 📐 2D/3D Geometry Calculator – C++

A console-based **C++ application** designed to calculate geometric properties of both **2D and 3D shapes**, including area, perimeter, surface area, and volume.

This project demonstrates structured programming, mathematical computation, and modular function design using C++.

---

## 🚀 Features

- 📏 Calculates **area and perimeter** of 2D shapes  
- 📦 Calculates **surface area and volume** of 3D shapes  
- 🔢 Supports multiple shapes including:
  - Square, Rectangle, Circle, Triangle, Ellipse  
  - Cube, Cuboid, Sphere, Cylinder, Cone  
- 🔄 Interactive user input system  
- 🧩 Modular function-based design  

---

## 🧠 Key Concepts Demonstrated

- Structured programming in C++

- Mathematical formula implementation

## 🛠️ Tech Stack

- **Language:** C++  
- **Libraries Used:** `<iostream>`, `<cmath>`  
- **Environment:** Visual Studio / Any C++ Compiler  

---

## ▶️ How to Run

1. Open the project in **Visual Studio** or any C++ IDE  
2. Compile and run the `main.cpp` file  
3. Follow on-screen instructions:
   - Choose between **2D or 3D shapes**
   - Select a shape
   - Enter required values  
4. View calculated results
